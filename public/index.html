AJAX examples

The following are examples of techniques to make requests of a web server from JavaScript code running in a client browser.

XMLHttp Request

The XmlHttpRequest is the original strategy to create AJAX requests, and is supported nearly universally web browsers. See the MDN article for details.

JQuery AJAX Request

The popular JQuery library wraps many core JavaScript library features with simplified syntax and convenience functions, including the creation and processing of AJAX requests. See the JQuery documentation for details.

Fetch API

The Fetch API is the newest standard for making HTTP requests from JavaScript. It utilizes Promises to streamline the processing of AJAX requests. See the MDN documentation for the Fetch API and the Promises API for more details.
